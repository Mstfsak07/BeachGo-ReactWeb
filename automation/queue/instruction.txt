## AdГ„Вұm 6: AuthService.cs

Eksik metodlarГ„Вұ implement et:

**ForgotPasswordAsync:**
```
1. Email ile kullanГ„ВұcГ„ВұyГ„Вұ bul ГўВҖВ” bulamazsan SUCCESS dГғВ¶n (security: email enumeration ГғВ¶nleme)
2. OtpService.GenerateTokenAsync(email, "PasswordReset") ГғВ§aГ„ВҹГ„Вұr
3. MockSmsService veya email service ile token'Г„Вұ gГғВ¶nder (log'a yaz mock iГғВ§in)
4. AuthResult { Success = true, Message = "Reset link sent" } dГғВ¶n
```

**ResetPasswordAsync:**
```
1. OtpService.ValidateTokenAsync(email, "PasswordReset", token) ГғВ§aГ„ВҹГ„Вұr
2. GeГғВ§ersizse AuthResult { Success = false, Message = "Invalid or expired token" }
3. GeГғВ§erliyse: kullanГ„ВұcГ„ВұnГ„Вұn password hash'ini gГғВјncelle
4. OtpService.InvalidateTokenAsync ile token'Г„Вұ geГғВ§ersiz kГ„Вұl
5. AuthResult { Success = true, Message = "Password reset successful" }
```

**VerifyEmailAsync:**
```
1. OtpService.ValidateTokenAsync(email, "EmailVerification", token) ГғВ§aГ„ВҹГ„Вұr
2. GeГғВ§ersizse: AuthResult { Success = false }
3. GeГғВ§erliyse: User.EmailConfirmed = true yap, kaydet
4. Token'Г„Вұ invalidate et
5. AuthResult { Success = true, Message = "Email verified" }
```

**ResendVerificationAsync:**
```
1. Email ile kullanГ„ВұcГ„ВұyГ„Вұ bul
2. EmailConfirmed zaten true ise: AuthResult { Success = false, Message = "Already verified" }
3. Yeni token ГғВјret, gГғВ¶nder
4. AuthResult { Success = true }
```
