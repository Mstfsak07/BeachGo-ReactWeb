## Yapılacak Değişiklikler

### 1. `src/api/axios.js`

Mevcut dosyayı oku, sonra şu yapıya göre güncelle:

- `baseURL`: `process.env.REACT_APP_API_URL || 'http://localhost:5000/api'`
- `refreshAccessToken` fonksiyonu: `export` edilmeli
- Request interceptor: localStorage'dan `accessToken` alıp `Authorization: Bearer <token>` header ekle
- Response interceptor: 401 gelirse `refreshAccessToken` çağır, yeni token ile isteği tekrarla; refresh başarısızsa localStorage temizle ve `/login`'e yönlendir

### 2. `src/context/AuthContext.jsx`

Mevcut dosyayı oku, sonra şu state ve fonksiyonların var olduğundan emin ol:

- `user` state (null başlangıç)
- `loading` state (true başlangıç, token kontrolü bitince false)
- `isAuthenticated` = `user !== null`
- `login(email, password)`: API çağrısı yap, `accessToken` + `refreshToken` localStorage'a kaydet, `user` set et
- `logout()`: localStorage'ı temizle, `user` null yap, `/login`'e yönlendir
- `register(name, email, password)`: API çağrısı yap
- `useEffect`: Uygulama başında localStorage'daki token kontrol et, geçerliyse user set et
- localStorage key'leri: `accessToken`, `refreshToken`, `user`

### 3. `src/pages/Login.jsx`

Mevcut dosyayı oku, sonra şu eklemeleri yap:

- Form validation: email format + password min 6 karakter
- `isLoading` state: submit sırasında buton disabled
- `error` state: API hatasını kırmızı alert kutusunda göster
- "Sifremi Unuttum" linki: `/forgot-password`
- Basarili login: `navigate('/home')` veya `navigate(from, { replace: true })`
- AuthContext'ten `login` fonksiyonunu kullan

### 4. `src/pages/Register.jsx`

Mevcut dosyayı oku, sonra şu eklemeleri yap:

- Validation: ad/soyad zorunlu, email format, password min 6, sifre tekrar eslesmesi
- `isLoading` state: submit sırasında "Kaydediliyor..." goster
- `error` state: API hataları
- Basarili kayit: "E-posta dogrulama linki gonderildi" mesaji

### 5. `src/components/PrivateRoute.jsx`

Dosya yoksa olustur, varsa gozden gecir:

- AuthContext'ten `user` ve `loading` al
- `loading === true`: spinner veya `<div>Yukleniyor...</div>` dondur
- `user === null`: `<Navigate to="/login" state={{ from: location }} replace />` dondur
- User varsa: `<Outlet />` dondur

### 6. `src/components/Navbar.jsx`

Mevcut dosyayı oku, sonra:

- `user` null ise: "Giris Yap" ve "Kayit Ol" butonları goster
- `user` varsa: kullanici adi, "Profilim", "Rezervasyonlarim", "Favorilerim", "Cikis" goster
- AuthContext'ten `user` ve `logout` al

### 7. `src/App.js`

Mevcut dosyayı oku, sonra route yapısını duzenle:

- Public route'lar: `/`, `/login`, `/register`, `/beaches/:id`, `/forgot-password`
- Korumalı route'lar (PrivateRoute ile sarılı): `/profile`, `/reservations`, `/favorites`
- AuthContext `loading` true iken `<div>Yukleniyor...</div>` goster
